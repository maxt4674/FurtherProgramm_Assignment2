REQUIRED DEPENDENCIES:
- Please have JAVA FX and JAVA FX SDK installed and present in the workspace
- Please input the slf4j-api and sqlite-jdbc files/folders as a referenced library in eclipse, with them also being put into the assignment_2 modulePath

COMPILATION AND RUN
- To compile, make sure the required dependencies are installed, and if eclipse is being used, just run the Main.java file under the src folder as your JRE System library should then automatically compile then run the application
